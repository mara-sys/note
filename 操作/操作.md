#二 软件使用
##2.1 markdown
###### 首行缩进
```
半方大的空白&ensp;或&#8194;
全方大的空白&emsp;或&#8195;
不断行的空白格&nbsp;或&#160;
```
注意：不要漏掉分号

###### 添加目录
1、[toc]
2、在vscode中安装插件`Markdown All in One`，安装后点开md文件，然后快捷键`CTRL+SHIFT+P`，输入`Markdown All in One: Create Table of Contents`，回车即可。

###### 生成侧边目录
1、安装nodejs
[nodejs下载地址](https://nodejs.org/en/)
2、测试是否安装成功
在cmd下输入npm -v查看版本号
3、安装插件
```shell
npm install -g i5ting_toc
```
4、在笔记.md文件下执行
```shell
i5ting_toc -f demo.md -o
```
